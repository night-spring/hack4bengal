## 🌟 Why AgriLink?

- <b>For Farmers:</b> Instantly list waste, get AI-powered price suggestions, earn new income, and track CO₂ savings. No tech skills required.
- <b>For Industry:</b> Discover, filter, and procure verified agricultural waste for sustainable production, with transparent supply chains and quality assurance.
- <b>For the Planet:</b> Reduces open burning, landfill waste, and tracks environmental impact in real time. Every transaction means less pollution and more carbon savings.

- 🔒 <b>Decentralized Login (Civic Auth):</b> Secure, privacy-first authentication for all users.

## 🚦 How It Works: End-to-End Flow

1. <b>Sign Up/Login:</b> Users authenticate securely via Civic Auth (decentralized, privacy-first).
2. <b>List Waste:</b> Farmers upload photos and details. Gemini AI classifies and suggests prices.
3. <b>Marketplace:</b> Listings go live. Industries browse, filter, and make offers.
4. <b>Smart Pricing:</b> Gemini API and market data ensure fair, dynamic pricing.
5. <b>Deal & Tokenization:</b> On agreement, Aptos blockchain mints tokens and records the transaction.
6. <b>Impact Tracking:</b> CO₂ savings and rewards are tracked in real time.
7. <b>Withdraw & Analytics:</b> Users manage earnings, see analytics, and download reports.

## 📦 Getting Started

### Frontend
```bash
cd frontend
npm install # or yarn install
# Set up .env.local with API keys and MongoDB URI
npm run dev
# Open http://localhost:3000
```

### Backend
```bash
cd backend
pip install -r requirements.txt
# Set up .env with Aptos private keys and config
python main.py
```
